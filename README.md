# CSE370_Smart_Semester_Planner

The Smart Semester Planner is an intelligent academic advising platform designed
to transform the traditional, manual course registration process into a streamlined,
data-driven decision-making experience. Built upon a robust relational database
architecture using PHP and MySQL with role based login system, the system goes
beyond basic data entry by enforcing strict academic integrity through a temporal
prerequisite validation engine and a recursive dependency graph manager. It serves
as an automated advisor by leveraging a constraint-based recommendation engine
that evaluates departmental degree deficits, historical academic performance, and
desired workload preferences to generate personalized, balanced semester plans.
By combining schema-level database failsafes with modular analytics such as
department-specific risk evaluation and intelligent retake prioritization, the platform
empowers students to navigate complex curriculum requirements efficiently, ensuring
steady progress toward graduation while actively minimizing the risk of academic
burnout.
